# hhsh Skill (能不能好好说话)

翻译中文互联网缩写和黑话，例如 yyds、awsl、kkp 等。

## 可用命令

### 查询缩写
`ybot hhsh guess "<text>"`

输入缩写文本，返回可能的翻译。支持同时查询多个词（空格分隔）。

示例:
- `ybot hhsh guess "yyds"` → yyds: 永远的神
- `ybot hhsh guess "awsl"` → awsl: 啊我死了
